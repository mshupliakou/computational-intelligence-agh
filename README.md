# Computational Intelligence - Laboratory Classes 🧠💻

This repository contains the source code, datasets, and final reports for the **Computational Intelligence** (Inteligencja Obliczeniowa) course. 

## 📌 About the Project

The purpose of this repository is to document the progress and results of various machine learning and artificial intelligence laboratory assignments. It covers data preprocessing, model training, hyperparameter tuning, and performance evaluation using different algorithms.

> **Note:** While the code and this documentation are in English, the official laboratory reports (`.pdf` and `.tex` files) are written in **Polish**.

## 📂 Repository Structure

The repository is organized by laboratory assignments. Each folder typically contains:
* Jupyter Notebooks (`.ipynb`) with the implementation.
* reports containing the LaTeX source files and the final PDF report in Polish.
